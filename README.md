# Desktop Pet (Godot)
<img width="64" height="64" alt="icon" src="https://github.com/user-attachments/assets/14aef0cc-667f-4de6-add2-b9567eb1ae60" />

(WIP) Godot implementation of a customizable Desktop Pet. Compatible with Windows, MacOS, and Linux (X11)!

## Capabilities
- Load pet with weighted randomized behavior.
- Pick up and throw pet around.
- Resizable/Repositional window overlay (Windows OS only).
